Obten un token de consulta para la API

  1. Obten tu token de acceso creando un usuario o iniciando sesión aquí
  2. El token que generes en el paso anterior tiene una vigencia de 4 horas. Recuerda solicitar uno nuevo después de ese tiempo.
  3. Con el token podrás conectarte a los endpoints disponibles

Accede o descarga la información

  1. Puedes consultar información de la API con el token que obtuviste en la sección anterior.
    1. Busca el valor de id_token
    2. Asignalo a tu petición un header con nombre Authorization
  2. Tambien puedes descargar directamente la información de los endpoints en formato JSON y CSV aquí
